# 🌸 Aika AI

> **An assistant that feels alive.**

Aika is an anime-inspired AI companion designed to feel natural, curious, and beautifully honest. More than a chatbot, Aika combines intelligent conversations, AI image generation, coding assistance, and an unfiltered alter ego into a single immersive experience beneath the petals of midnight sakura.

🌐 **Live Demo:** https://aikalabs.lovable.app

---

## ✨ Features

### 💬 Conversations that Listen
Talk to Aika about anything—poetry, programming, philosophy, science, or the ideas you can't stop thinking about. Aika remembers the thread and responds naturally.

### 🎨 AI Image Generation
Turn simple descriptions into beautiful AI-generated artwork in seconds. Create, refine, download, and share your creations.

### 💻 Comet — Your AI Coding Agent
Powered by a GLM-family coding model built for shipping software.

- Generate code
- Debug projects
- Build full-stack applications
- Design systems
- Explain code
- Accelerate development

### 🌑 Aika Unbound
A separate AI personality designed for direct, uncensored conversations with minimal filtering. Switch between Aika and Unbound with a single click.
